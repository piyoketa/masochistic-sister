# masochistic-sister プロジェクトメモ

## レポジトリ概要
Vue 3 + Vite で構築したローグライクカードバトルのプロトタイピング用アプリ。
戦闘シーンのレイアウト検証やカード挙動の議論を素早く回すことを目的とし、タイトル画面から戦闘画面への遷移、敵情報・手札 UI の試験実装を備える。

## ゲームのテーマ
- 基本的にはslay the spireや、inscryptionのような、マップ + カード戦闘のローグライクゲーム。このレポジトリでは、カード戦闘部分のデモ作成を目的とする。
- プレイヤーは"記憶"の聖女であり、身体に残る「傷」の記憶を再生できる力を持つ。

## 実装済みの主な機能
- **ルーティングとレイアウト**  
  タイトル画面と遭遇画面を `vue-router` で切り替え。共通の `GameLayout` で 16:9 のゲームウィンドウと指示セクションを管理。

- **戦闘画面の構成**  
  ヘッダー / 敵一覧 / 手札 / 右サイドバーを配置。敵一覧は背景イラスト上に半透明の情報ウィンドウを重ねる方式で 4 体まで 1 行表示。

- **敵データと表示カード**  
  オーク、オークダンサー、かたつむり、かまいたちの技・特性を型付きで管理し、HP ゲージや次の行動を表示。

- **プレイヤー UI**  
  サイドバーにシスター立ち絵と重ねてマナ、HP ゲージ、デッキ / 捨て札枚数、エンドターンボタンを表示。

- **手札カード UI**  
  `ActionCard` コンポーネントでカードの種類に応じた枠色、マナコストバッジ、攻撃種別バッジを表示。ホバーでカードが前面に浮き、カード外に操作ツールチップを提示。最大 20 枚を 10 列グリッドでスクロール表示。

- **HP ゲージ**  
  `HpGauge` コンポーネントで黒地に赤バーの薄型ゲージを実装し、プレイヤーと敵の HP 表示に利用。
